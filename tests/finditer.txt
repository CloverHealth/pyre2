Simple tests for the ``finditer`` function.
===========================================

    >>> import re2
    >>> re2.set_fallback_notification(re2.FALLBACK_EXCEPTION)

    >>> len(list(re2.finditer(r'\w+', open("cnn_homepage.dat").read())))
    14230

    >>> [m.group(1) for m in re2.finditer(r'\n#hdr-editions(.*?)\n', open("cnn_homepage.dat").read())]
    [' a { text-decoration:none; }', ' li { padding:0 10px; }', ' ul li.no-pad-left span { font-size:12px; }']

    >>> [m.group(1) for m in re2.finditer(r'^#hdr-editions(.*?)$',
    ... open("cnn_homepage.dat").read(), re2.M)]
    [' a { text-decoration:none; }', ' li { padding:0 10px; }', ' ul li.no-pad-left span { font-size:12px; }']
